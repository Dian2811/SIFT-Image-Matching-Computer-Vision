# Image Matching Menggunakan Algoritma SIFT

Proyek ini adalah Final Project mata kuliah Visi Komputer yang bertujuan mengimplementasikan algoritma Scale-Invariant Feature Transform (SIFT) untuk mencocokkan fitur pada dua citra dengan perbedaan rotasi dan perspektif.

## Fitur Utama
- Deteksi keypoints dan ekstraksi descriptor menggunakan SIFT.
- Feature matching menggunakan Brute-Force Matcher.
- Penerapan Lowe's Ratio Test untuk akurasi hasil.

## Hasil Eksperimen
Hasil Matching Ditemukan 541 titik kecocokan valid pada objek sampul buku.

## Cara Menjalankan
1. Clone repositori ini.
2. Pastikan library OpenCV, NumPy, dan Matplotlib terpasang.
3. Jalankan file.
